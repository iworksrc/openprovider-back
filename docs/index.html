    func TribonacсiIteroBig ¶ 

    func TribonacсiIteroBig(argument int) *big.Int

    Calculates the value of the nth member of the Triboacci series. An iterative calculation algorithm is used. To store the calculation results, use type of data big.Int is not a bounding dimension (the size of a variable of type big.Int is limited only the size of the machine's RAM). As a consequence, the function can calculate the values of the series from rather large values of the argument (one million or more) for acceptable time.
